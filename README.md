## Golang
Miscellaneous apps I am developing using the Go programming language (otherwise known as "Golang"). 

**How to Install Go**
```bash
sudo apt update
sudo apt install golang -y
```

**If/Else Statement**
```go
package main
import "fmt"
func main() {
    if 1 + 1 == 2 {
        fmt.Println("True")
    } else {
        fmt.Println("False")
    }
}
```
